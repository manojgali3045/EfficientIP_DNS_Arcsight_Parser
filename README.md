# EfficientIP DNS ArcSight Flex Parser

This repository contains **work‑in‑progress (WIP)** development of an ArcSight
**FlexConnector (sdkrfilereader) regex parser** for **EfficientIP DNS (BIND/named) syslog logs**.

⚠️ **Disclaimer**  
- This parser is **NOT production‑ready**
- It is shared **for learning, experimentation, and community collaboration**
- No vendor support is implied

## Scope
- DNS client query logs
- DNS answer logs
- BIND / named style syslog output
- Parsing for ArcSight SmartConnector (8.x)

## Status
🚧 Under active development  
✅ Community feedback and contributions welcome

## Notes
- No real customer data is included
- All examples must be sanitized before use
- Parser behavior may vary depending on EfficientIP configuration
